Bootcamp Helper

Extensão exemplo Chrome (Manifest V3). Popup + Service Worker + Content Script.

O que faz

Tem um popup que envia um PING para o service worker, abre uma página de documentação do Chrome e um content script que destaca links na página https://developer.chrome.com/*.

Instalação manual

  1. Baixe o pacote (.zip) ou clone o repositório.
  2. Abra chrome://extensions no Chrome e ative Developer mode.
  3. Clique em Load unpacked e selecione a pasta my-chrome-extension.
